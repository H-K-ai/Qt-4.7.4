# Qt-4.7.4

近几年做过的一部分Qt项目。

管理系统类项目涵盖了Qt连接数据库三种类的用法，各种基础控件的使用，文件的操作等。项目太多不一一介绍。

挑选几个代表例子简单介绍。

##catchFish

包含使用QGraphicsScene，QGraphicsItem制作动画效果实现捕鱼游戏。

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/catchFish.jpg)


##colorDialog

模拟了一个拾色器模块。

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/colorDialog.jpg)


##express

包含使用QNetworkAccessManager调用网站API得到结果并解析的物流查询系统。

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/express.jpg)


##fingerprint

包含使用QAxBase，QAxWidget调用Windows下ActiveX控件实现指纹识别等功能。

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/fingerprint.jpg)


##movieplayer

包含使用QProcess调用MPlayer实现视频播放的功能。


##musicPlayer

包含使用Phonon播放音乐文件的功能。


##QExcel

包含使用QAxObject调用Excel.Application实现Qt对Excel文件的操作。


##smartHotel

包含使用QIODevice实现在Windows下的串口通讯。开源库为QextSerialPort

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/smartHotel.jpg)


##wanmeiyichu

包含使用QStateMachine，QSS实现了大量的界面动画以及控件美化效果。

![image](https://github.com/llwslc/Qt-4.7.4/blob/master/Screenshots/wanmeiyichu.jpg)
